About
============
This module useful to generate receipt from odoo backend. You can also send the receipt by email. This module is very useful to generate a pos receipt from the odoo backend pos order. You can also send the receipt by email to the customer easily.
Installation
============
1) Copy module files to addon folder.
2) Restart odoo service (sudo service odoo-server restart).
3) Go to your odoo instance and open apps (make sure to activate debug mode).
4) click on update app list. 
5) search module name and hit install button.
